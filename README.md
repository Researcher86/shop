# shop

Интернет-магазин по продаже компьютерной техники. 

### Технологии: 
- Java 8
- Spring (MVC, Security, Data (JPA/Hibernate), Test) 
- JSP
- Maven 
- SLF4J (Logback) 
- CSS (Twitter Bootstrap) 
- JS (AJAX, JQuery)
- Tomcat
- MySQL
- HSQLDB
